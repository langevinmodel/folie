folie.OverdampedBF¶

class folie.OverdampedBF(basis, **kwargs)[source]¶

A class that implement a overdamped model with basis function

Base model for overdamped Langevin equations, defined by

dX(t) = mu(X,t)dt + sigma(X,t)dW_t

Parameters:

has_exact_density – bool, set to true if an exact density is implemented
